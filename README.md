# 📊 Restaurant Data Analysis (Internship Project)

📌 Internship Project | Real-world dataset | End-to-end Data Analysis  

This project was completed during my **Data Analyst Internship at Cognifyz Technologies**, where I worked on analyzing a real-world restaurant dataset to uncover patterns in customer behavior, pricing, and cuisine trends.

---

## 🔍 Project Objective

The goal of this project was to perform **Exploratory Data Analysis (EDA)** to:
- Understand customer preferences  
- Analyze pricing patterns  
- Identify trends in cuisines and ratings  
- Generate insights for data-driven decision-making  

---

## ⚙️ Tech Stack

- **Python** (Pandas, NumPy)  
- **Matplotlib**  
- **Jupyter Notebook**  

---

## 🧹 Data Cleaning & Preprocessing

- Handled missing values and null entries  
- Removed duplicate records  
- Standardized inconsistent data formats  
- Prepared dataset for accurate and efficient analysis  

---

## 📈 Exploratory Data Analysis

- Analyzed distribution of different cuisines  
- Compared **Movies vs TV Shows equivalent?** ❌ (remove if not relevant)  
- Studied pricing trends across restaurants  
- Examined customer ratings and their relationship with pricing  
- Identified top-performing restaurant categories  

---

## 💡 Key Insights

- Popular cuisines vary significantly based on customer demand  
- Mid-range priced restaurants tend to receive higher ratings  
- Strong relationship observed between pricing and customer satisfaction  
- Certain categories dominate the restaurant market  

---

## 📊 Visualizations

- Bar charts for cuisine distribution  
- Scatter plots for price vs ratings  
- Trend analysis using graphs  

---

## 🚀 Outcome

This project demonstrates how raw data can be transformed into **meaningful insights** to support business decisions such as pricing strategies, customer targeting, and service improvement.
